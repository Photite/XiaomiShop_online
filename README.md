# **XiaomiShop_online**

本项目是自学VUE和SpringBoot后，自己感兴趣实现的前后端分离的VUE项目，本项目为小米商城的前端部分

主要功能：

首页商品展示功能

全部商品功能

模糊搜索功能

购物车功能

收藏夹功能

订单功能

本项目使用支付宝沙箱，实现了模拟支付。
